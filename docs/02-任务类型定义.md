├── app              APP 逆向
│   ├── android      Android (APK)

## 2.2 工单格式 (Job)

每进入一个 stage 创建一个 Job 实例：

```json
{
  "job_id": "web-sig-20260422-001",
  "task_id": "web-sig-taobao-20260422",
  "stage": "js_locate",
  "type": "web.js.signature",
  "input_fingerprint": "sha256(...)",
  "executor_type": "codex",
  "owner": null,
  "status": "pending",
  "priority": "normal",
  "created_at": "2026-04-22T10:00:00Z",
  "started_at": null,
  "finished_at": null,
  "result_paths": {
    "result_json": null,
    "log_txt": null,
    "artifacts": []
  },
  "error": null,
  "retry_count": 0,
  "max_retries": 3
}
```

### Job 状态机

```
pending → running → verified (done)
                  → failed → retry (if retry_count < max)
                           → blocked (记录到 SESSION-STATE.md)
                  → skipped (被控制器判定跳过)
```

### 判重规则

```python
# 同一 (stage, input_fingerprint) 只能存在一个未完成 job
def is_duplicate(stage, fingerprint):
    existing = read_jobs(f"tasks/queue/{stage}.jobs.jsonl")
    return any(
        j["stage"] == stage
        and j["input_fingerprint"] == fingerprint
        and j["status"] not in ("verified", "failed")
        for j in existing
    )
```

## 2.3 任务下发格式

### Web 逆向任务

```json
{
  "task_id": "web-sig-taobao-20260422",
  "type": "web.js.signature",
  "priority": "high",
  "target": {
    "url": "https://taobao.com/api/search",
    "method": "POST",
    "body_pattern": "{\"keyword\":\"手机\"}",
    "known_params": ["appid", "timestamp", "sig"]
  },
  "actions": [
    {"type": "click", "selector": "#search-input"},
    {"type": "type", "selector": "#search-input", "value": "手机"},
    {"type": "click", "selector": "#search-btn"}
  ],
  "constraints": {
    "max_steps": 20,
    "timeout_minutes": 30
  },
  "notes": "目标站点有重度反调试，需要先过 debugger 检测"
}
```

### APP 逆向任务

```json
{
  "task_id": "app-dy-android-20260422",
  "type": "app.android.signature",
  "priority": "high",
  "target": {
    "package": "com.ss.android.ugc.aweme",
    "apk_path": "/tools/apk/tiktok-v30.apk",
    "target_api": "/aweme/v1/feed/"
  },
  "hook_targets": [
    "com.ss.android.common.apk.util.SignUtil",
    "com.bytedance.frameworks.baselib.network"
  ],
  "capture_options": {
    "mitm_enabled": true,
    "ssl_pinning_bypass": true,
    "methods": ["onRequest", "getSign"]
  }
}
```

## 2.4 任务优先级

| 优先级 | 值 | 使用场景 |
|--------|----|---------|
| critical | 0 | Blocker 修复、紧急项目 |
| high | 1 | 主线项目、高价值目标 |
| normal | 2 | 一般逆向任务 |
| low | 3 | 试探性研究、工具优化 |
